# PF2 Portfolio

Vue 기반 포트폴리오를 React + Vite + TypeScript로 점진 전환한 프로젝트입니다.

## 시작하기
```bash
npm install
npm run dev
```

## 스크립트
```bash
npm run dev        # 개발 서버
npm run build      # 프로덕션 빌드
npm run preview    # 빌드 결과 미리보기
npm run lint       # ESLint 검사
npm run test       # Vitest 실행
```
